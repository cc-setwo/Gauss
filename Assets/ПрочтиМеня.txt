Программа создана для решения матриц, которые изучаются на курсе линейной алгебры.
Примеры для проверки программы можно взять: https://math.dartmouth.edu/archive/m23s06/public_html/handouts/row_reduction_examples.pdf
Если в длину будет ровно 3 строки, то можно использовать стандартный режим разделяя значения запятой. Если другое число, то нужно использовать MxN режим.
И разделять строки через знак '!'. А значения также через запятые.
Рассмотрим первый пример из сайта:
1 5 7
-2 -7 -5
Запись в программе будет выглядеть(в режиме MxN): 1,5,7!-2,-7,-5. Для вычисления нажать кнопку "Calculate".
Второй пример:
0 2 1 -8
1 -2 -3 0
-1 1 2 3
Так как 3 строки в длину, то можно использовать стандартный режим. Запись примера в программе будет выглядеть следующим образом:
0,2,1,-8
1,-2,-3,0
-1,1,2,3
Для вычисления нажать кнопку "Calculate".
